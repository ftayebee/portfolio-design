The Challenge

A Marketplace That Had to Scale Fast

ShopEdge needed a fully featured multi-vendor marketplace — one that could onboard hundreds of independent sellers, handle concurrent orders, automate payouts, and remain fast under load. The existing WooCommerce setup was crumbling under 3,000+ SKUs.

01

Vendor Isolation & Permission System

Each seller needed a sandboxed dashboard with role-based access — preventing cross-vendor data leakage while sharing a single platform codebase.

02

Real-time Inventory Sync

Product availability had to update instantly across all storefronts without polling — requiring WebSockets and an optimistic UI strategy in Vue.js.

03

Automated Multi-party Payouts

Stripe Connect had to split each transaction automatically between the platform, the vendor, and the delivery fee — with full reconciliation reports.

04

Performance at Scale

Lazy loading, query optimisation, Redis caching, and CDN asset delivery reduced page load from 6.4s to under 1.2s at launch.

The Solution

How I Built It

A modular Laravel monolith with Vue.js SPA frontend — designed for long-term maintainability and rapid feature iteration.

Modular Domain Architecture

Each business concern (Vendors, Orders, Payouts, Reviews) lives in its own Laravel service class — isolated, testable, and independently deployable.

Vue.js Reactive Storefront

Pinia state management + Vue Router for a seamless SPA experience. Optimistic UI updates make inventory changes feel instant to shoppers.

Stripe Connect Payouts

Automated split payments with configurable platform fees, instant vendor payouts, chargeback handling, and a full admin reconciliation dashboard.

Redis + Query Caching

Product catalogue and search results cached in Redis with tag-based invalidation — zero stale data, sub-100ms API responses under peak load.

Role-based Access Control

Spatie Permissions with custom middleware enforces vendor sandboxing — no vendor can read or modify another vendor's data, even at the API level.

Full Test Coverage

PHPUnit feature tests cover every critical path — checkout, payout calculation, vendor onboarding, and order state transitions — at 94% coverage.
